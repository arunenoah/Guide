🚀 Getting Started with Claude

Setup, configuration, and integration with your development workflow

1. Claude Code Installation & Setup

Install Claude Code CLI

For macOS: 
brew install anthropics/claude/claude-code
For Linux/Windows WSL: 
curl -fsSL https://install.claudecode.com | bash

Authenticate

Initialize Claude Code: 
claude-code
This will prompt you to authenticate with your Anthropic API key.
ℹ️ API Key Setup: Get your API key from https://console.anthropic.com. Store it securely as an environment variable.

Verify Installation

claude-code --version
claude-code --help

2. Basic Workflow

Step Command Description
1. Start in project directory cd /path/to/project Navigate to your codebase
2. Launch Claude Code claude-code Start interactive session
3. Ask questions Type naturally Describe what you need help with
4. Claude makes changes Review in editor Approve/reject changes via UI
5. Commit changes git commit Use /commit skill or manual git

3. Essential Commands & Skills

Slash Commands in Claude Code

Command Purpose
/help Get help about Claude Code features
/commit Generate and create a git commit with your changes
/review-pr Review a GitHub pull request
/fast Toggle fast mode for quicker responses
/file [path] Focus Claude's context on a specific file
/clear Clear conversation history
✓ Pro Tip: Use /commit to automatically generate well-formatted commit messages following your project conventions.

4. Project Configuration File

Create CLAUDE.md in your project root

{
"projectName": "Your SaaS App",
"description": "React + Laravel + Node microservices",
"techStack": {
  "frontend": ["React", "TypeScript", "Tailwind CSS"],
  "backend": ["Laravel", "PHP"],
  "services": ["Node.js", "AWS Lambda"],
  "database": ["PostgreSQL", "Redis"],
  "testing": ["Jest", "Playwright", "PHPUnit"]
},
"conventions": {
  "commits": "Use conventional commits (feat:, fix:, docs:)",
  "branches": "feature/*, bugfix/*, hotfix/*",
  "testing": "Write tests for all new features",
  "codeStyle": "Follow ESLint + Laravel PSR-12"
},
"importantFiles": [
  "src/components/",
  "app/Http/Controllers/",
  "tests/",
  ".github/workflows/"
]
}
ℹ️ Note: This file helps Claude understand your project structure and conventions automatically.

9. Common Issues & Troubleshooting

API Key Not Recognized

# Solution: Re-authenticate
claude-code --logout
claude-code  # Log in again

Model Issues

# Use the latest model
CLAUDE_MODEL=claude-opus-4-6 claude-code

Git Integration Not Working

# Ensure you're in a git repository
git init
git config user.name "Your Name"
git config user.email "your@email.com"
