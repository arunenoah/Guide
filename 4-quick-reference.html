React Common Patterns

📦 useState Hook

const [state, setState] = useState(initialValue)

Manage component state in functional components

🔄 useEffect Hook

useEffect(() => { /* effect */ }, [dependencies])

Handle side effects (API calls, subscriptions)

🎣 useCallback Hook

const memoized = useCallback(() => {}, [deps])

Memoize function to prevent unnecessary re-renders

💾 useContext Hook

const value = useContext(MyContext)

Access context value in component

🎯 useReducer Hook

const [state, dispatch] = useReducer(reducer, init)

Complex state management in components

🎭 Conditional Render

{condition ? <ComponentA /> : <ComponentB />}

Render different components based on condition

Laravel Common Patterns

📝 Create Model

php artisan make:model Post -mrc

Create Model with migration, controller, and resource

🔗 Eloquent Relationships

// One-to-Many
public function posts() {
    return $this->hasMany(Post::class);
}

Define relationships between models

✅ Form Validation

$request->validate([
    'email' => 'required|email|unique:users'
]);

Validate incoming request data

🔐 Authorization

$this->authorize('update', $post);
// or Gate::authorize('name')

Check user permissions

📊 Query Builder

User::where('active', 1)
    ->orderBy('created_at', 'desc')
    ->paginate(15)

Build complex queries

🔍 Eager Loading

$users = User::with('posts', 'comments')->get();

Prevent N+1 query problems
