code-reviewer-quality-skill:
  "## Third-Party API Integration Security
   Check for each external API:
   - [ ] Credentials in environment variables
   - [ ] API keys/secrets never logged
   - [ ] Proper authentication method
   - [ ] Timeout specified
   - [ ] Retry logic with exponential backoff
   - [ ] Error handling catches API exceptions
   - [ ] Webhook signatures verified (if applicable)
   - [ ] Rate limiting respected"

security-audit-skill:
  "## External Service Integration Security
   
   For ANY third-party service:
   
   ### Authentication & Credentials
   - [ ] API keys/credentials in environment variables
   - [ ] Credentials never logged
   - [ ] Proper auth method (OAuth, API Key, etc.)
   - [ ] Token expiration handled
   
   ### Error Handling
   - [ ] Specific exception types
   - [ ] API error details never exposed
   - [ ] Internal logging with full details
   - [ ] Graceful degradation
   - [ ] Retry logic with backoff
   
   ### Webhook Security (if applicable)
   - [ ] Signature verification
   - [ ] Timestamp validation
   - [ ] Idempotency keys
   
   ### Cloud Service Specific
   - [ ] IAM/RBAC properly configured
   - [ ] Encryption at rest
   - [ ] Encryption in transit
   - [ ] Audit trails enabled"
